# Skysport
# 💰 Трекер расходов

Мобильное веб-приложение для учёта расходов с аналитикой и синхронизацией с Google Sheets.

## 🚀 Быстрый старт

1. Скачай `index.html`
2. Открой в браузере — работает без сервера
3. Добавь на главный экран телефона: **Share → Add to Home Screen**

---

## ☁️ Синхронизация с Google Sheets

### Шаг 1: Создай Google Таблицу
Открой [sheets.google.com](https://sheets.google.com) и создай новую таблицу.  
В первую строку добавь заголовки (необязательно, скрипт сам заполнит):
```
Дата | Статья расхода | Сумма | Категория | Тип | Заметка
```

### Шаг 2: Добавь Apps Script
1. В таблице: **Расширения → Apps Script**
2. Удали весь существующий код
3. В приложении открой **⚙️ → Google Sheets → "Скопировать Apps Script"**
4. Вставь скопированный код в редактор и нажми **💾 Сохранить**

### Шаг 3: Развернуть как веб-приложение
1. Нажми **Развернуть → Новое развёртывание**
2. Тип: **Веб-приложение**
3. Описание: любое (например "Расходы")
4. Выполнять от имени: **Я (your@email.com)**
5. Доступ: **Все** ← обязательно!
6. Нажми **Развернуть** и скопируй URL

### Шаг 4: Подключи к приложению
В приложении: **⚙️ → Google Sheets → вставь URL скрипта**

Теперь кнопка **☁️** в шапке будет отправлять все несинхронизированные записи в таблицу.

